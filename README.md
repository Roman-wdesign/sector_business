# sector\_business

### Features:

* Data from API
* table (grid)
* search
* sort list on @click (A-Z, Z-A, default)
* Pagination without plugins
* Pagination (router-link, url change, change active paginated page @click)
* Paginated buttons (\<router-link>)



## Project setup

```
npm install
```
